## How to Use

1. Open your project
2. Move the **playhead** to the frame you want
3. On menu bar → **Frame → Copy Current Frame** or usining shortcur **Command-Shift-C**
4. Paste it anywhere with **⌘ + V**

---

## Use Cases

- Create thumbnails
- Share UI snapshots
- Add visuals to docs or tweets
- Quickly extract assets without exporting video
